üîó Step 2: Publish Your Sheet

2.1 Publish to Web

  1. In your Google Sheet, go to File ‚Üí Share ‚Üí Publish to web
  2. In the dialog box:
    • Link type: Select "Web page"
    • Sheet: Select "Sheet1" (or your events sheet)
    • Format: Select "Comma-separated values (.csv)"
  3. Click "Publish"
  4. Copy the generated URL (it should look like: https://docs.google.com/spreadsheets/d/YOUR_SHEET_ID/pub?output=csv)

2.2 Share the Sheet

Make sure your sheet is accessible:

  1. Click the "Share" button in the top-right corner
  2. Set permissions to "Anyone with the link can view"
  3. Click "Done"
